Component: ScrollShadow
Library: heroui
Category: Data display
Source URL: https://heroui.com/docs/react/components/scroll-shadow

Overview:
Apply visual shadows to indicate scrollable content overflow with automatic detection of scroll position.
Apply visual shadows to indicate scrollable content overflow with automatic detection of scroll position.

Props Description:

orientation
The scroll direction
Type:"vertical" | "horizontal"
Default:"vertical"

variant
The visual shadow effect style
Type:"fade"
Default:"fade"

size
The shadow gradient size in pixels
Type:number
Default:40

offset
The scroll offset before showing shadows (in pixels)
Type:number
Default:0

hideScrollBar
Whether to hide the native scrollbar
Type:boolean
Default:false

isEnabled
Whether scroll shadow detection is enabled
Type:boolean
Default:true

visibility
Controlled shadow visibility state
Type:"auto" | "both" | "top" | "bottom" | "left" | "right" | "none"
Default:"auto"

className
Additional CSS classes to apply to the root element
Type:string
Default:-

children
The scrollable content
Type:ReactNode
Default:-

Events:

onVisibilityChange
Callback invoked when shadow visibility changes
Type:(visibility: ScrollShadowVisibility) => void
