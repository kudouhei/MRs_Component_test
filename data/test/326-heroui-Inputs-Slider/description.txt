Component: Slider
Library: heroui
Category: Inputs
Source URL: https://heroui.com/docs/react/components/slider

Overview:
A slider allows a user to select one or more values within a range
A slider allows a user to select one or more values within a range

Props Description:

value
The current value (controlled)
Type:number | number[]
Default:-

defaultValue
The default value (uncontrolled)
Type:number | number[]
Default:-

minValue
The slider's minimum value
Type:number
Default:0

maxValue
The slider's maximum value
Type:number
Default:100

step
The slider's step value
Type:number
Default:1

formatOptions
The display format of the value label
Type:Intl.NumberFormatOptions
Default:-

orientation
The orientation of the slider
Type:"horizontal" | "vertical"
Default:"horizontal"

isDisabled
Whether the slider is disabled
Type:boolean
Default:-

aria-label
Accessibility label for the slider
Type:string
Default:-

aria-labelledby
ID of element that labels the slider
Type:string
Default:-

className
Additional CSS classes
Type:string
Default:-

children
Slider content or render function
Type:ReactNode | RenderFunction
Default:-

render
Overrides the default DOM element with a custom render function.
Type:DOMRenderFunction<keyof React.JSX.IntrinsicElements, SliderRenderProps>
Default:-

style
Inline styles
Type:CSSProperties
Default:-

index
Index of the thumb within the slider
Type:number
Default:0

name
The name of the input element, used when submitting an HTML form
Type:string
Default:-

state
The state of the slider
Type:SliderState

values
Values managed by the slider by thumb index
Type:number[]

getThumbValueLabel
Returns the string label for the specified thumb's value
Type:(index: number) => string

Events:

onChange
Handler called when the value changes
Type:(value: number | number[]) => void

onChangeEnd
Handler called when dragging ends
Type:(value: number | number[]) => void
