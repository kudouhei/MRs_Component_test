Component: Anchor
Library: ant-design
Category: Navigation
Source URL: https://ant.design/components/anchor/

Overview:
Hyperlinks to scroll on one page.
For displaying anchor hyperlinks on page and jumping between them.

Props Description:

affix
Fixed mode of Anchor
Type:boolean | Omit<AffixProps, 'offsetTop' | 'target' | 'children'>
Default:true

bounds
Bounding distance of anchor area
Type:number
Default:5

classNames
Customize class for each semantic structure inside the component. Supports object or function.
Type:Record< SemanticDOM , string> | (info: { props })=> Record< SemanticDOM , string>
Default:-

getContainer
Scrolling container
Type:() => HTMLElement
Default:() => window

getCurrentAnchor
Customize the anchor highlight
Type:(activeLink: string) => string
Default:-

offsetTop
Pixels to offset from top when calculating position of scroll
Type:number
Default:0

showInkInFixed
Whether show ink-square when affix={false}
Type:boolean
Default:false

styles
Customize inline style for each semantic structure inside the component. Supports object or function.
Type:Record< SemanticDOM , CSSProperties> | (info: { props })=> Record< SemanticDOM , CSSProperties>
Default:-

targetOffset
Anchor scroll offset, default as offsetTop , example
Type:number
Default:-

items
Data configuration option content, support nesting through children
Type:{ key, href, title, target, children }[] see
Default:-

direction
Set Anchor direction
Type:vertical | horizontal
Default:vertical

replace
Replace items' href in browser history instead of pushing it
Type:boolean
Default:false

key
The unique identifier of the Anchor Link
Type:string | number
Default:-

href
The target of hyperlink
Type:string

target
Specifies where to display the linked URL
Type:string

title
The content of hyperlink
Type:ReactNode

children
Nested Anchor Link, Attention: This attribute does not support horizontal orientation
Type:AnchorItem []
Default:-

Events:

onChange
Listening for anchor link change
Type:(currentActiveLink: string) => void

onClick
Set the handler to handle click event
Type:(e: MouseEvent, link: object) => void
