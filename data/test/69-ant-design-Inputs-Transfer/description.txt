Component: Transfer
Library: ant-design
Category: Inputs
Source URL: https://ant.design/components/transfer/

Overview:
Double column transfer choice box.
Transfer the elements between two columns in an intuitive and efficient way.

Props Description:

actions
A set of operations that are sorted from top to bottom. When an array of strings is provided, default buttons will be used; when an array of ReactNode is provided, custom elements will be used
Type:ReactNode[]
Default:[ > , < ]

classNames
Customize class for each semantic structure inside the component. Supports object or function.
Type:Record< SemanticDOM , string> | (info: { props })=> Record< SemanticDOM , string>
Default:-

dataSource
Used for setting the source data. The elements that are part of this array will be present the left column. Except the elements whose keys are included in targetKeys prop
Type:RecordType extends TransferItem = TransferItem []
Default:[]

disabled
Whether disabled transfer
Type:boolean
Default:false

selectionsIcon
custom dropdown icon
Type:React.ReactNode

filterOption
A function to determine whether an item should show in search result list, only works when searching, (add direction support since 5.9.0+)
Type:(inputValue, option, direction: left | right ): boolean
Default:-

A function used for rendering the footer
Type:(props, { direction }) => ReactNode
Default:-

A custom CSS style used for rendering the transfer columns. Use styles.section instead
Type:object | ({direction: left | right }) => object
Default:-

locale
The i18n text including filter, empty text, item unit, etc
Type:{ itemUnit: string; itemsUnit: string; searchPlaceholder: string; notFoundContent: ReactNode | ReactNode[]; }
Default:{ itemUnit: item , itemsUnit: items , notFoundContent: The list is empty , searchPlaceholder: Search here }

operations
A set of operations that are sorted from top to bottom. Use actions instead.
Type:string[]
Default:[ > , < ]

operationStyle
A custom CSS style used for rendering the operations column. Use styles.actions instead.
Type:object
Default:-

pagination
Use pagination. Not work in render props
Type:boolean | { pageSize: number, simple: boolean, showSizeChanger?: boolean, showLessItems?: boolean }
Default:false

render
The function to generate the item shown on a column. Based on an record (element of the dataSource array), this function should return a React element which is generated from that record. Also, it can return a plain object with value and label , label is a React element and value is for title
Type:(record) => ReactNode
Default:-

selectAllLabels

Type:string[] | number[]
