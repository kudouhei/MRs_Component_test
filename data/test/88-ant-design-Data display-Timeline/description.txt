Component: Timeline
Library: ant-design
Category: Data display
Source URL: https://ant.design/components/timeline/

Overview:
Vertical display timeline.
Node supports loading to indicate loading, and reverse property to control the order of nodes.

Props Description:

Type:Items []
Default:-

Type:start | alternate | end
Default:start

Type:vertical | horizontal
Default:vertical

Type:ReactNode
Default:false
