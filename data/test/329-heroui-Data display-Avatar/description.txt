Component: Avatar
Library: heroui
Category: Data display
Source URL: https://heroui.com/docs/react/components/avatar

Overview:
Display user profile images with customizable fallback content
Display user profile images with customizable fallback content

Props Description:

size
Avatar size
Type:'sm' | 'md' | 'lg'
Default:'md'

color
Fallback color theme
Type:'default' | 'accent' | 'success' | 'warning' | 'danger'
Default:'default'

variant
Visual style variant
Type:'default' | 'soft'
Default:'default'

className
Additional CSS classes
Type:string
Default:-

src
Image source URL
Type:string
Default:-

srcSet
The image srcset attribute for responsive images
Type:string
Default:-

sizes
The image sizes attribute for responsive images
Type:string
Default:-

alt
Alternative text for the image
Type:string
Default:-

crossOrigin
CORS setting for the image request
Type:'anonymous' | 'use-credentials'
Default:-

loading
Native lazy loading attribute
Type:'eager' | 'lazy'
Default:-

delayMs
Delay before showing fallback (prevents flash)
Type:number
Default:-

Events:

onLoad
Callback when the image loads successfully
Type:(event: SyntheticEvent<HTMLImageElement>) => void

onError
Callback when there's an error loading the image
Type:(event: SyntheticEvent<HTMLImageElement>) => void
