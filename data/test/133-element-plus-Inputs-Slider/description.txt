Component: Slider
Library: element-plus
Category: Inputs
Source URL: https://element-plus.org/en-US/component/slider

Overview:
A Vue 3 based component library for designers and developers

Props Description:

model-value / v-model
binding value
Type:number / array
Default:0

min
minimum value
Type:number
Default:0

max
maximum value
Type:number
Default:100

disabled
whether Slider is disabled
Type:boolean
Default:false

step
step size, can be a number or 'mark' 2.13.6 to restrict values to marks. When set to 'mark' , the marks attribute must be set
Type:number / string
Default:1

show-input
whether to display an input box, works when range is false and step is not 'mark'
Type:boolean
Default:false

show-input-controls
whether to display control buttons when show-input is true
Type:boolean
Default:true

size
size of the slider wrapper, will not work in vertical mode
Type:enum
Default:default

input-size
size of the input box, when set size , the default is the value of size
Type:enum
Default:default

show-stops
whether to display breakpoints
Type:boolean
Default:false

show-tooltip
whether to display tooltip value
Type:boolean
Default:true

format-tooltip
format to display tooltip value
Type:Function
Default:—

range
whether to select a range
Type:boolean
Default:false

vertical
vertical mode
Type:boolean
Default:false

height
slider height, required in vertical mode
Type:string
Default:—

aria-label a11y 2.7.2
native aria-label attribute
Type:string
Default:—

range-start-label
when range is true, screen reader label for the start of the range
Type:string
Default:—

range-end-label
when range is true, screen reader label for the end of the range
Type:string
Default:—

format-value-text
format to display the aria-valuenow attribute for screen readers
Type:Function
Default:—

tooltip-class
custom class name for the tooltip
Type:string
Default:—

placement
position of Tooltip
Type:enum
Default:top

marks
marks, type of key must be number and must in closed interval [min, max] , each mark can custom style
Type:object
Default:—

persistent 2.9.5
when slider tooltip inactive and persistent is false , tooltip will be destroyed. persistent always be false when show-tooltip is false
Type:boolean
Default:true

label a11y deprecated
native aria-label attribute
Type:string
Default:—

change
triggers when the value changes (if the mouse is being dragged, this event only fires when the mouse is released)
Type:Function

input
triggers when the data changes (It'll be emitted in real time during sliding)
Type:Function

Events:

validate-event
whether to trigger form validation
Type:boolean
