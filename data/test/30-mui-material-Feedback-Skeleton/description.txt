Skeleton API
API reference docs for the React Skeleton component. Learn about the props, CSS, and other APIs of this exported module.
For examples and details on the usage of this React component, visit the component demo pages:

Props Description:

animation
The animation. If false the animation effect is disabled.
Type:'pulse' | 'wave' | false
Default:'pulse'

children
Optional children to infer width and height from.
Type:node
Default:-

classes
Override or extend the styles applied to the component. See CSS classes API below for more details.
Type:object
Default:-

component
The component used for the root node. Either a string to use a HTML element or a component.
Type:elementType
Default:-

height
Height of the skeleton. Useful when you don't want to adapt the skeleton to a text element but for instance a card.
Type:number | string
Default:-

sx
The system prop that allows defining system overrides as well as additional CSS styles. See the `sx` page for more details.
Type:Array<func | object | bool> | func | object
Default:-

variant
The type of content that will be rendered.
Type:'circular' | 'rectangular' | 'rounded' | 'text' | string
Default:'text'

width
Width of the skeleton. Useful when the skeleton is inside an inline element with no width of its own.
Type:number | string
Default:-
