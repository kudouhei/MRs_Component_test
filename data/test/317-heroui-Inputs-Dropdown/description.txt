Component: Dropdown
Library: heroui
Category: Inputs
Source URL: https://heroui.com/docs/react/components/dropdown

Overview:
A dropdown displays a list of actions or options that a user can choose
A dropdown displays a list of actions or options that a user can choose

Props Description:

isOpen
Sets the open state of the menu (controlled)
Type:boolean
Default:-

defaultOpen
Sets the default open state of the menu (uncontrolled)
Type:boolean
Default:-

trigger
The type of interaction that triggers the menu
Type:"press" | "longPress"
Default:"press"

className
Additional CSS classes
Type:string
Default:-

children
Dropdown content
Type:ReactNode
Default:-

placement
Placement of the popover relative to the trigger
Type:"bottom" | "bottom left" | "bottom right" | "bottom start" | "bottom end" | "top" | "top left" | "top right" | "top start" | "top end" | "left" | "left top" | "left bottom" | "start" | "start top" | "start bottom" | "right" | "right top" | "right bottom" | "end" | "end top" | "end bottom"
Default:"bottom"

selectionMode
Whether single or multiple selection is enabled
Type:"single" | "multiple" | "none"
Default:"none"

selectedKeys
The currently selected keys (controlled)
Type:Iterable<Key>
Default:-

defaultSelectedKeys
The initial selected keys (uncontrolled)
Type:Iterable<Key>
Default:-

disabledKeys
Keys of disabled items
Type:Iterable<Key>
Default:-

id
Unique identifier for the item
Type:Key
Default:-

textValue
Text content of the item for typeahead
Type:string
Default:-

variant
Visual variant of the item
Type:"default" | "danger"
Default:"default"

type
Type of indicator to display
Type:"checkmark" | "dot"
Default:"checkmark"

isSelected
Whether the item is selected
Type:boolean

isIndeterminate
Whether the item is in an indeterminate state
Type:boolean

isFocused
Whether the item is focused
Type:boolean

isDisabled
Whether the item is disabled
Type:boolean

isPressed
Whether the item is being pressed
Type:boolean

Events:

onOpenChange
Handler called when the open state changes
Type:(isOpen: boolean) => void

onSelectionChange
Handler called when the selection changes
Type:(keys: Selection) => void

onAction
Handler called when an item is activated
Type:(key: Key) => void
