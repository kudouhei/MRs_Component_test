Stepper API
API reference docs for the React Stepper component. Learn about the props, CSS, and other APIs of this exported module.
For examples and details on the usage of this React component, visit the component demo pages:

Props Description:

activeStep
Set the active step (zero based index). Set to -1 to disable all the steps.
Type:integer
Default:0

alternativeLabel
If set to 'true' and orientation is horizontal, then the step label will be positioned under the icon.
Type:bool
Default:false

children
Two or more <Step /> components.
Type:node
Default:-

classes
Override or extend the styles applied to the component. See CSS classes API below for more details.
Type:object
Default:-

component
The component used for the root node. Either a string to use a HTML element or a component.
Type:elementType
Default:-

connector
An element to be placed between each step.
Type:element
Default:<StepConnector />

nonLinear
If set the Stepper will not assist in controlling steps for linear flow.
Type:bool
Default:false

orientation
The component orientation (layout flow direction).
Type:'horizontal' | 'vertical'
Default:'horizontal'

sx
The system prop that allows defining system overrides as well as additional CSS styles. See the `sx` page for more details.
Type:Array<func | object | bool> | func | object
Default:-
