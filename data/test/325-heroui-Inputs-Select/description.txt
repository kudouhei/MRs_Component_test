Component: Select
Library: heroui
Category: Inputs
Source URL: https://heroui.com/docs/react/components/select

Overview:
A select displays a collapsible list of options and allows a user to select one of them
A select displays a collapsible list of options and allows a user to select one of them

Props Description:

placeholder
Temporary text that occupies the select when it is empty
Type:string
Default:'Select an item'

selectionMode
Whether single or multiple selection is enabled
Type:"single" | "multiple"
Default:"single"

isOpen
Sets the open state of the menu (controlled)
Type:boolean
Default:-

defaultOpen
Sets the default open state of the menu (uncontrolled)
Type:boolean
Default:-

disabledKeys
Keys of disabled items
Type:Iterable<Key>
Default:-

isDisabled
Whether the select is disabled
Type:boolean
Default:-

value
Current value (controlled)
Type:Key | Key[] | null
Default:-

defaultValue
Default value (uncontrolled)
Type:Key | Key[] | null
Default:-

isRequired
Whether user input is required
Type:boolean
Default:-

isInvalid
Whether the select value is invalid
Type:boolean
Default:-

name
The name of the input, used when submitting an HTML form
Type:string
Default:-

autoComplete
Describes the type of autocomplete functionality
Type:string
Default:-

fullWidth
Whether the select should take full width of its container
Type:boolean
Default:false

variant
Visual variant of the component. primary is the default style with shadow. secondary is a lower emphasis variant without shadow, suitable for use in surfaces.
Type:"primary" | "secondary"
Default:"primary"

className
Additional CSS classes
Type:string
Default:-

children
Select content or render function
Type:ReactNode | RenderFunction
Default:-

render
Overrides the default DOM element with a custom render function.
Type:DOMRenderFunction<keyof React.JSX.IntrinsicElements, SelectRenderProps>
Default:-

placement
Placement of the popover relative to the trigger
Type:"bottom" | "bottom left" | "bottom right" | "bottom start" | "bottom end" | "top" | "top left" | "top right" | "top start" | "top end" | "left" | "left top" | "left bottom" | "start" | "start top" | "start bottom" | "right" | "right top" | "right bottom" | "end" | "end top" | "end bottom"
Default:"bottom"

defaultChildren
The default rendered value
Type:ReactNode

isPlaceholder
Whether the value is a placeholder
Type:boolean

state
The state of the select
Type:SelectState

selectedItems
The currently selected items
Type:Node[]

Events:

onOpenChange
Handler called when the open state changes
Type:(isOpen: boolean) => void

onChange
Handler called when the value changes
Type:(value: Key | Key[] | null) => void
