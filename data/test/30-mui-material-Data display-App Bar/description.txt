AppBar API
API reference docs for the React AppBar component. Learn about the props, CSS, and other APIs of this exported module.

Props Description:

children
The content of the component.
Type:node
Default:-

classes
Override or extend the styles applied to the component. See CSS classes API below for more details.
Type:object
Default:-

color
The color of the component. It supports both default and custom theme colors, which can be added as shown in the palette customization guide .
Type:'default' | 'inherit' | 'primary' | 'secondary' | 'transparent' | 'error' | 'info' | 'success' | 'warning' | string
Default:'primary'

elevation
Shadow depth, corresponds to dp in the spec. It accepts values between 0 and 24 inclusive.
Type:number
Default:4

enableColorOnDark
If true, the color prop is applied in dark mode.
Type:bool
Default:false

position
The positioning type. The behavior of the different options is described in the MDN web docs . Note: sticky is not universally supported and will fall back to static when unavailable.
Type:'absolute' | 'fixed' | 'relative' | 'static' | 'sticky'
Default:'fixed'

square
If false , rounded corners are enabled.
Type:bool
Default:true

sx
The system prop that allows defining system overrides as well as additional CSS styles. See the `sx` page for more details.
Type:Array<func | object | bool> | func | object
Default:-
