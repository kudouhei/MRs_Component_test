Component: Rating
Library: ant-design
Category: Inputs
Source URL: https://ant.design/components/rate/

Overview:
Used for rating operation on something.
Replace the default star to other character like alphabet, digit, iconfont or even Chinese word.

Default:<StarFilled />

Default:-

Type:number
Default:5

defaultValue
The default value
Type:number
Default:0

disabled

Type:boolean
Default:false

keyboard
Support keyboard operation
Type:boolean
Default:true

size
Star size
Type:'small' | 'medium' | 'large'
Default:'medium'

style

Type:CSSProperties
Default:-

Default:-

value
The current value

Default:-
