Component: Button
Library: heroui
Category: Inputs
Source URL: https://heroui.com/docs/react/components/button

Overview:
A clickable button component with multiple variants and states
A clickable button component with multiple variants and states

Props Description:

variant
Visual style variant
Type:'primary' | 'secondary' | 'tertiary' | 'outline' | 'ghost' | 'danger'
Default:'primary'

size
Size of the button
Type:'sm' | 'md' | 'lg'
Default:'md'

fullWidth
Whether the button should take full width of its container
Type:boolean
Default:false

isDisabled
Whether the button is disabled
Type:boolean
Default:false

isPending
Whether the button is in a loading state
Type:boolean
Default:false

isIconOnly
Whether the button contains only an icon
Type:boolean
Default:false

children
Button content or render prop
Type:React.ReactNode | (values: ButtonRenderProps) => React.ReactNode
Default:-

render
Overrides the default DOM element with a custom render function.
Type:DOMRenderFunction<keyof React.JSX.IntrinsicElements, ButtonRenderProps>
Default:-

isPressed
Whether the button is currently pressed
Type:boolean

isHovered
Whether the button is hovered
Type:boolean

isFocused
Whether the button is focused
Type:boolean

isFocusVisible
Whether the button should show focus indicator
Type:boolean

Events:

onPress
Handler called when the button is pressed
Type:(e: PressEvent) => void
